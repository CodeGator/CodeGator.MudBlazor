#### What does it do?
This package contains MudBlazor derived controls that are used by various CodeGator solutions.

#### What platform(s) does it support?
* .NET 10.x or higher

#### How do I install it?
The binary is hosted on [NuGet](https://www.nuget.org/packages/Codegator/). To install the package using the NuGet package manager:

PM> Install-Package CodeGator.MudBlazor
